# 4.1 / 4.2 文件名改动说明

## 改动了什么

- **4.1**：将原乱码文件名 `4.1-妯′豢瀛︿範.md` 重命名为 **`4.1-模仿学习.md`**。
- **4.2**：将原乱码文件名 `4.2-寮哄寲瀛︿範.md` 重命名为 **`4.2-强化学习.md`**。

文件内容未做修改，仅修正了文件名。`docs/4-classical/README.md` 中已使用正确中文链接（`4.1-模仿学习.md`、`4.2-强化学习.md`），本次改名后目录内链接即可正常跳转。

## 有什么用

1. **消除乱码**：原文件名是编码错误导致的乱码（如“妯′豢瀛︿範”应为“模仿学习”，“寮哄寲瀛︿範”应为“强化学习”），修正后文件名与章节含义一致，便于识别。
2. **链接可用**：README 中的「📖 阅读」链接指向的是正确中文文件名，此前实际文件名为乱码，链接会 404；改名后点击即可打开对应文档。
3. **跨平台与协作**：使用正确 UTF-8 中文文件名，在不同系统与 Git 协作时更不易出现编码问题，克隆/检出后文件名显示一致。
4. **文档结构清晰**：与 4.3、4.4 等命名风格统一（数字 + 中文主题），方便维护和查找。
