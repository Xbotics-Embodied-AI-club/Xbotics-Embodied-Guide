# STE-VLN：基于事件中心知识图谱的视觉-语言导航增强

*基于 ArXiv 机器人学每日速递原文的整理，仅保留核心内容，供学习参考。*

---

## 1. 背景与问题

VLN 智能体在未见环境中的长视野推理与粗粒度指令理解不足，现有知识图谱方法未充分利用源自人类情节记忆的多模态事件知识。

## 2. 方法要点

- **YE-KG**：首个大规模多模态时空知识图谱，约 86k 节点、83k 边，从真实室内视频中用多模态 LLM（LLaVA、GPT-4）将非结构化视频流抽取为结构化语义-动作-效果事件。
- **STE-VLN**：通过粗到细分层检索将知识图谱集成到 VLN 模型，使智能体检索因果事件序列并与自中心视觉观测动态融合。

## 3. 结果与意义

在 REVERIE、R2R、R2R-CE 上超越 SOTA。多模态时空知识图谱可成为长视野导航推理的重要基础设施。

**关键词**：Vision-Language Navigation · Knowledge Graph · Episodic Memory · REVERIE · R2R
