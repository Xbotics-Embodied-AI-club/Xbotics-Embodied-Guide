## 2. 编辑点评：今日最值得关注的三个方向

1. **具身智能数据瓶颈的新解法** — AoE 系统用人类自身 + 手机作为「数据生产者」，降低第一视角交互数据采集成本；若规模化成功，将对具身基础模型训练产生深远影响。
2. **人形机器人运动能力的双重突破** — OmniTrack 与 OmniXtreme 分别从「物理可行性解耦」和「流匹配扩展容量」突破保真度-可扩展性权衡；一小时连续追踪与翻转侧手翻标志着运动控制进入新阶段。
3. **VLA 架构设计的四维进化** — FAVLA（力觉快慢解耦）、StemVLA（4D 时空表征）、VCA（点击替代语言）、ABPolicy（B 样条异步推理）覆盖感知融合、世界建模、交互界面、输出平滑，VLA 生态从「能用」走向「好用」。
