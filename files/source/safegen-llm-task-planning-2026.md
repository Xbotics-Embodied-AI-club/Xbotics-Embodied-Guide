# SafeGen-LLM：具备安全泛化能力的大语言模型任务规划框架

*基于 ArXiv 机器人学每日速递原文的整理，仅保留核心内容，供学习参考。*

---

## 1. 背景与问题

安全关键任务规划面临：经典规划器扩展性差、RL 泛化弱、基础 LLM 无法保证安全性。

## 2. 方法要点

- **基准**：构建含显式安全约束的多域 PDDL3 基准。
- **两阶段后训练**：先在约束遵从规划数据上 SFT，学习规划语法与语义；再用形式化验证驱动的细粒度奖励机引导 GRPO 做安全对齐，结合课程学习处理复杂任务。

## 3. 结果与意义

在多域规划任务与多种输入格式下超越前沿闭源基线，安全泛化能力更强。用可验证的形式安全约束 LLM 规划器，是走向安全关键机器人系统真实部署的重要一步。
