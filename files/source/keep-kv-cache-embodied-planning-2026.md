# KEEP：面向高效具身规划的 KV 缓存中心记忆管理

*基于 ArXiv 机器人学每日速递原文的整理，仅保留核心内容，供学习参考。*

---

## 1. 背景与问题

记忆增强型 LLM 在复杂长视野具身规划中表现好，但以原始文本存记忆导致提示过长、预填充延迟高；KV 缓存复用又因频繁更新而效益大减。

## 2. 方法要点

- **静态-动态记忆构建**：混合粒度记忆分组，减少 KV 缓存重计算。
- **多跳记忆重计算**：动态识别不同记忆组间的重要交叉注意力，迭代重建记忆交互。
- **层平衡记忆加载**：消除不同层间不均衡的 KV 缓存加载。

## 3. 结果与意义

在 ALFRED 上实现约 2.68× 加速且精度几乎无损，首 token 延迟降低约 1.9×，成功率较 CacheBlend 提升约 4.13%。被 DAC 2026 录用。为具身智能的「持久记忆 + 快速检索」提供了高效工程方案。
